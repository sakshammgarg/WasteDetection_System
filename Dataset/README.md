The dataset used in this project is sourced from Kaggle.

Dataset: Waste Classification Data
Link: https://www.kaggle.com/datasets/techsash/waste-classification-data

The dataset contains labeled images of two waste categories: organic (O) and recyclable (R). 
It is designed to support the training and evaluation of machine learning models for image-based waste classification. 
The objective is to accurately classify each input image into one of these two predefined categories.
